      .. rubric:: Description

      The onemkl::rng::gumbel class object is used in the
      onemkl::rng::generate function to provide Gumbel distributed random
      numbers with displacement (a) and scalefactor (``b``, β), where a,
      β∈\ ``R`` ; β > 0.
